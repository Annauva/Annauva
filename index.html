Hello!

I’m Anna Grape; a web designer and an illustrator living just outside of Stockholm, Sweden. I’ve been working as an illustrator for 7 years but now I want to try something new.
So I decided to learn web design! It has been a very fun and challenging journey and I still learn new stuff every day.
My dream right now is to find a fun workplace who can see the potential in my illustrator background and that I can use all my skills to create beautiful websites!
